# 🌿 Ambreen & Inamur's Wedding Invitation

A beautiful, animated wedding invitation website built with Next.js 14, featuring a sophisticated pista (pistachio green) color palette and smooth scroll animations.

## ✨ Features

### 🎨 Design & Aesthetics
- **Pista Color Palette**: Sophisticated green-based theme with cream and gold accents
- **Islamic Elements**: Arabic calligraphy, Quranic verses, and traditional Islamic wedding aesthetics
- **Google Fonts**: Playfair Display, Dancing Script, Cormorant Garamond, Inter, and Amiri (Arabic)
- **Glass-morphism Effects**: Modern backdrop-blur styling throughout

### 🎭 Animations & Interactions
- **Framer Motion**: Smooth, professional animations throughout the site
- **Scroll-triggered Animations**: Content reveals as you scroll with intersection observer
- **Floating Elements**: Subtle animated decorative elements
- **Interactive Countdown**: Real-time countdown to the wedding date (April 22, 2026)
- **Hover Effects**: Elegant card lift effects and interactive buttons

### 📱 Responsive Design
- **Mobile-first**: Optimized for all device sizes (320px - 1440px+)
- **Breakpoints**: Mobile, Tablet, Laptop, Desktop
- **Touch-friendly**: 48px minimum touch targets for mobile devices
- **Performance**: Optimized animations and loading for all devices

### 🕌 Cultural Integration
- **Islamic Wedding Structure**: Proper ceremony chronology (Haldi, Mehndi, Nikah, Rukhsati)
- **Arabic Text**: Bismillah, Quranic verses, and wedding duas
- **Traditional Colors**: Respectful use of Islamic green with elegant design
- **Family Honor**: Proper display of both families and grandparents

## 🚀 Getting Started

### Prerequisites
- Node.js 18.0.0 or higher
- npm or yarn package manager

### Installation

1. **Clone or navigate to the project directory:**
   ```bash
   cd /Users/apple/projects/invitation_card
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the wedding invitation.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run type-check` - Run TypeScript check

## 🛠️ Technology Stack

### Core Framework
- **Next.js 14** - React framework with App Router
- **React 18** - UI library with latest features
- **TypeScript** - Type-safe development

### Styling & Animations
- **Tailwind CSS 3.4** - Utility-first CSS framework
- **Framer Motion 10** - Production-ready motion library
- **Custom CSS** - Advanced animations and effects

### UI Components
- **Lucide React** - Beautiful icon library
- **Radix UI** - Headless UI primitives
- **React Hook Form** - Form handling with Zod validation

### Performance & Optimization
- **Next.js Image Optimization** - Automatic image optimization
- **Font Optimization** - Google Fonts with display=swap
- **Static Generation** - Pre-rendered pages for optimal performance

## 🔧 Customization

### Updating Wedding Details
Edit `/src/app/page.tsx` to modify:
- Couple names and family information
- Event dates and times
- Venue information
- Contact details

### Changing Colors
Update `/tailwind.config.js` and `/src/styles/globals.css` to modify the color scheme.

### Adding Sections
Create new components in `/src/components/sections/` and import them into the main page.
